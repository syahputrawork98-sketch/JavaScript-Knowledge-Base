# BK-01: Active Proposals (Stage 1-3)

> **"Radar Inovasi Hub."**

Buku ini memantau proposal-proposal aktif yang sedang dikembangkan oleh TC39. Ini adalah tempat di mana arsitek mendeteksi perubahan besar yang akan datang pada sirkuit JavaScript.

---

## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: Incubation (Stage 1-2)](./CH-01_Incubation/)**
  - Konsep awal: Records and Tuples, Pattern Matching, Async Context.
- **[CH-02: Near Completion (Stage 3)](./CH-02_NearCompletion/)**
  - Siap diimplementasi: Decorators, Temporal, ShadowRealm.

## 🎯 Fokus Pembelajaran
1. Memahami desain filosofi di balik proposal baru (mengapa fitur ini dibutuhkan?).
2. Mencoba fitur melalui Babel/Polyfills bahkan sebelum rilis resmi.
3. Mengenali risiko penggunaan fitur Stage < 3 di lingkungan produksi.
