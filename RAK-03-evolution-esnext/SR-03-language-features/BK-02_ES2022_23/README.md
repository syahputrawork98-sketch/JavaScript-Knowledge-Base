# BK-02: Recent Advancement (ES2022 - ES2023)

> **"Penguatan Struktur dan Ergonomi Hub."**

Buku ini membedah kemajuan di tahun 2022 dan 2023, memperkenalkan enkapsulasi tingkat tinggi melalui field privat dan cara baru memanipulasi Array tanpa merusak data asli.

---

## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: Top Level Await and New Built-ins](./CH-01_TopLevelAwait/)**
  - Eksekusi asinkron di root modul dan `Array.at()`.
- **[CH-02: Class Private and Change Array by Copy](./CH-02_ClassPrivateStatic/)**
  - #PrivateFields dan metode `toSorted`, `toReversed`, `toSpliced`.

---

## 🎯 Fokus Pembelajaran
1. Memahami kekuatan `Top-level await` dalam menyederhanakan inisialisasi modul.
2. Menguasai prinsip "Immutability" melalui metode `Change Array by Copy`.
3. Mengenali `Object.hasOwn` sebagai standar baru yang lebih aman daripada `hasOwnProperty`.
