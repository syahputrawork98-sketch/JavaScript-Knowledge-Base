## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: Advanced Grouping and Strings](./CH-01_ObjectGrouping/)**

## 🎯 Fokus Pembelajaran
1. Memahami kemudahan kategorisasi data melalui `groupBy`.
2. Mengenali perbaikan fundamental pada RegExp melalui `v` flag (set notation).
3. Bersiap untuk transisi dari `Date` ke `Temporal` sebagai standar waktu Hub.
