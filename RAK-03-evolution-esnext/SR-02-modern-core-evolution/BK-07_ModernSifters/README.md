# BK-07: Modern Sifters (ES2024+)

> **"Penyaring Data Generasi Baru."**

Buku ini mengarsipkan fitur-fitur ES2024+ yang berfokus pada manipulasi data non-destruktif dan pengelompokan koleksi yang lebih ekspresif.

---

## Struktur Bab (Gold Standard)

- **[CH-01: Array By Copy](./CH-01_ArrayByCopy/)**: Metode non-mutating seperti `toSorted`, `toReversed`, `toSpliced`, dan `with`.
- **[CH-02: Object Grouping](./CH-02_ObjectGrouping/)**: `Object.groupBy()` dan `Map.groupBy()` untuk klasifikasi data tingkat tinggi.

---

## Fokus Pembelajaran
1. Mengadopsi manipulasi array yang aman tanpa efek samping.
2. Memahami grouping APIs sebagai pengganti pola `reduce` manual yang berulang.
3. Menangkap arah desain JavaScript modern yang makin deklaratif.
