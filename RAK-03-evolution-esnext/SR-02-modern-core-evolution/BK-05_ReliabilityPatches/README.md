# BK-05: Reliability Patches (ES2020 - ES2021)

> **"Pondasi Ketahanan Arsitektur Modern."**

Buku ini mengarsipkan fitur-fitur ES2020 dan ES2021 yang memperkuat keandalan aliran data dan ergonomi ekspresi JavaScript modern.

---

## Struktur Bab (Gold Standard)

- **[CH-01: Safety Valves](./CH-01_SafetyValves/)**: Optional Chaining dan Nullish Coalescing untuk akses data yang lebih aman.
- **[CH-02: Logical Assignment](./CH-02_LogicalAssignment/)**: Logical assignment, numeric separators, dan pola pembaruan nilai yang lebih presisi.

---

## Fokus Pembelajaran
1. Menguasai operator pengaman untuk data nullish dan akses properti mendalam.
2. Memahami penyederhanaan ekspresi assignment tanpa kehilangan kejelasan logika.
3. Membaca fase awal gelombang fitur reliabilitas yang membentuk JavaScript modern.
