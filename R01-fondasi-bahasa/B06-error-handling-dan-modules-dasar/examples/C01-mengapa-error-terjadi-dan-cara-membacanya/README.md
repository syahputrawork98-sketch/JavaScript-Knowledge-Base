# C01 Examples

Contoh untuk chapter `C01-mengapa-error-terjadi-dan-cara-membacanya.md`.

## File

- `example.js`: membaca error sederhana saat variabel yang dipakai belum ada.
- `example-02.js`: membaca bagian penting dari object error dengan `try...catch`.
- `example-03.js`: membandingkan pola membaca nama error, pesan, dan lokasi umum masalah.

## Cara Menjalankan

```bash
node example.js
node example-02.js
node example-03.js
```
