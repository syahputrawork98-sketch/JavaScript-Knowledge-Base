# C07 Examples

Contoh untuk chapter `C07-mengapa-modules-diperlukan.md`.

## File

- `example.js`: satu file kecil yang memanggil beberapa fungsi terkait.

## Cara Menjalankan

```bash
node example.js
node example-02.js
node example-03.js
```
