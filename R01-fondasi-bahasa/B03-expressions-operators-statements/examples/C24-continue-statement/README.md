# Example C24

Contoh runnable untuk bab **C24 - continue Statement**.

## Tujuan Example

Folder ini menunjukkan cara `continue` melewati sisa iterasi saat kondisi tertentu terpenuhi.

## Daftar File

- `example.js` menunjukkan `continue` pada `for` dan `while`.
- `example-02.js` fokus pada `continue` di `for` loop.
- `example-03.js` menunjukkan `continue` di `while` dengan update counter yang tetap aman.

## Poin Penting

- `continue` tidak menghentikan loop, hanya melewati sisa body iterasi saat ini.
- Pada `while`, update state harus tetap hati-hati agar tidak membuat loop bermasalah.
- `continue` cocok untuk skip cepat pada data yang tidak relevan.

## Jalankan

- `node example.js`
- `node example-02.js`
- `node example-03.js`
