# Example C16

Contoh runnable untuk bab **C16 - Comma Operator**.

## Tujuan Example

Folder ini membantu pembaca membedakan comma operator sebagai expression dari koma biasa sebagai separator syntax.

## Daftar File

- `example.js` menunjukkan hasil akhir comma operator dan membedakannya dari koma separator array.
- `example-02.js` menunjukkan bahwa comma operator mengevaluasi beberapa expression dan mengembalikan yang terakhir.
- `example-03.js` menunjukkan alternatif yang lebih mudah dibaca.

## Poin Penting

- Comma operator mengevaluasi dari kiri ke kanan, lalu mengembalikan nilai terakhir.
- Operator ini valid, tetapi sering mengurangi keterbacaan.
- Di banyak kasus, menulis statement terpisah lebih baik.

## Jalankan

- `node example.js`
- `node example-02.js`
- `node example-03.js`
