# Example C29

Contoh runnable untuk bab **C29 - Labelled Statements**.

## Tujuan Example

Folder ini memperlihatkan label pada loop untuk `break` atau `continue` ke level luar.

## Daftar File

- `example.js` menunjukkan `break` ke label loop luar.
- `example-02.js` menunjukkan `continue` dengan label.
- `example-03.js` menegaskan pentingnya nama label yang jelas.

## Poin Penting

- Label paling sering dipakai pada nested loop.
- `break label` dan `continue label` membantu mengontrol loop luar.
- Karena jarang dipakai, naming label harus jelas agar tetap terbaca.

## Jalankan

- `node example.js`
- `node example-02.js`
- `node example-03.js`
