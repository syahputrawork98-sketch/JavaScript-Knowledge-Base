# Example C18

Contoh runnable untuk bab **C18 - Block Statement**.

## Tujuan Example

Folder ini menunjukkan fungsi block `{}` sebagai pengelompokan statement dan pembatas scope.

## Daftar File

- `example.js` menunjukkan block biasa, block pada `if`, dan block di loop.
- `example-02.js` menegaskan block scope dengan `const` dan `let`.
- `example-03.js` menunjukkan block untuk variable sementara.

## Poin Penting

- Block membantu struktur baca dan juga membatasi scope `let`/`const`.
- Loop dan `if` sangat sering memakai block.
- Block kecil berguna untuk membuat variable sementara tidak bocor keluar.

## Jalankan

- `node example.js`
- `node example-02.js`
- `node example-03.js`
