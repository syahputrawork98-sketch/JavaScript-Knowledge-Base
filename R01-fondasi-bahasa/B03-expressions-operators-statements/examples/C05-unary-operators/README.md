# Example C05

Contoh runnable untuk bab **C05 - Unary Operators**.

## Tujuan Example

Folder ini memperlihatkan unary operator yang umum: unary plus/minus, logical not, `typeof`, `void`, dan `delete`.

## Daftar File

- `example.js` merangkum banyak unary operator dalam satu tempat.
- `example-02.js` menyorot `typeof`, `void`, dan `delete` pada object.
- `example-03.js` menunjukkan unary plus sebagai konversi number cepat.

## Poin Penting

- Unary plus sering dipakai untuk coercion ke number.
- `void` selalu menghasilkan `undefined`.
- `delete` menghapus property object, bukan nilai primitive lokal biasa.

## Jalankan

- `node example.js`
- `node example-02.js`
- `node example-03.js`
