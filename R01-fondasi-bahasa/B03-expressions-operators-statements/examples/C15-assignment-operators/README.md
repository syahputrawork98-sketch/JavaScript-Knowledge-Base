# Example C15

Contoh runnable untuk bab **C15 - Assignment Operators**.

## Tujuan Example

Folder ini memperlihatkan assignment biasa, compound assignment, dan destructuring assignment.

## Daftar File

- `example.js` menunjukkan compound assignment, assignment expression, dan destructuring dasar.
- `example-02.js` fokus pada compound assignment.
- `example-03.js` fokus pada destructuring assignment.

## Poin Penting

- Assignment sendiri adalah expression dan bisa menghasilkan nilai.
- Compound assignment meringkas pola update variabel.
- Destructuring assignment membantu mengambil nilai dari array/object dengan ringkas.

## Jalankan

- `node example.js`
- `node example-02.js`
- `node example-03.js`
