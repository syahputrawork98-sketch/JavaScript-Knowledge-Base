# Example C23

Contoh runnable untuk bab **C23 - Iteration Statements**.

## Tujuan Example

Folder ini memperlihatkan berbagai bentuk loop dasar di JavaScript dan kapan masing-masing cocok dipakai.

## Daftar File

- `example.js` merangkum `for`, `while`, `do...while`, `for...of`, dan `for...in`.
- `example-02.js` fokus pada `while` dan `do...while`.
- `example-03.js` membandingkan `for...of` dan `for...in`.

## Poin Penting

- `for` cocok saat pola iterasinya jelas dari awal.
- `do...while` selalu jalan minimal satu kali.
- `for...of` membaca nilai, sedangkan `for...in` membaca key/index enumerable.

## Jalankan

- `node example.js`
- `node example-02.js`
- `node example-03.js`
