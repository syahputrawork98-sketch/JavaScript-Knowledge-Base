# Docs B03

Dokumen ini mengatur aturan khusus buku B03.

## Tujuan Buku

- Pembaca sasaran: pemula JavaScript level fondasi.
- Outcome belajar: menguasai expressions, operators, dan statements secara operasional.
- Prasyarat: sudah menuntaskan `B01` dan `B02`.

## Struktur Konten Buku

- Folder materi utama: `chapters/`
- Folder contoh runnable: `examples/`
- Pola urutan bab: dari expression dasar, operator, lalu control flow.
- Pola latihan: checkpoint singkat per bab + contoh pembanding.
- Pola visual: secukupnya untuk relasi konsep yang sering membingungkan pemula.

## Catatan Operasional
