# C13 Examples

Contoh untuk chapter `C13-math-dan-number-builtins-praktis.md`.

## File

- `example.js`: pembulatan dengan `Math.floor()`, `Math.ceil()`, dan `Math.round()`.
- `example-02.js`: perbandingan angka dan nilai acak sederhana dengan `Math`.
- `example-03.js`: validasi dasar dengan `Number.isNaN()` dan `Number.isInteger()`.

## Cara Menjalankan

```bash
node example.js
node example-02.js
node example-03.js
```
