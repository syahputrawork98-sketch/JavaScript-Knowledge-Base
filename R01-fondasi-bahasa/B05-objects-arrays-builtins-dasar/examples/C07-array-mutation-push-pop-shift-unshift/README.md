# C07 Examples

Contoh untuk chapter `C07-array-mutation-push-pop-shift-unshift.md`.

## File

- `example.js`: menambah elemen ke belakang dengan `push()`.
- `example-02.js`: mengambil elemen terakhir dengan `pop()`.
- `example-03.js`: menambah dan mengambil elemen dari depan dengan `unshift()` dan `shift()`.

## Cara Menjalankan

```bash
node example.js
node example-02.js
node example-03.js
```
