# Index Bab B05

`B05` membangun fondasi struktur data inti JavaScript sebelum pembaca masuk ke error handling, modularisasi, dan object model yang lebih dalam.

## Daftar Bab (C01-C16)

1. `C01-object-literal-dan-property-access.md`  
   Tujuan: memahami object literal dan dua cara utama mengakses property.
2. `C02-object-mutation-method-dan-delete-dasar.md`  
   Tujuan: memahami perubahan isi object, method dasar, dan penggunaan `delete` secara sadar.
3. `C03-method-dasar-dan-this-awal.md`  
   Tujuan: memahami method sebagai function di dalam object dan peran awal `this`.
4. `C04-pemeriksaan-property-dan-iterasi-object-dasar.md`  
   Tujuan: memahami cara mengecek keberadaan property dan membaca isi object dengan aman.
5. `C05-array-literal-index-length-dan-update.md`  
   Tujuan: memahami array sebagai daftar berindeks dan konsekuensi `length`.
6. `C06-iterasi-array-dasar-dengan-loop-dan-foreach.md`  
   Tujuan: memahami cara membaca isi array secara berurutan.
7. `C07-array-mutation-push-pop-shift-unshift.md`  
   Tujuan: memahami operasi mutable paling umum pada array.
8. `C08-array-map-filter-dan-find-dasar.md`  
   Tujuan: memahami transformasi dan pencarian data array level pemula.
9. `C09-mutable-vs-non-mutable-operation-dasar.md`  
   Tujuan: membedakan operasi yang mengubah data asli dan operasi yang menghasilkan data baru.
10. `C10-built-in-object-keys-values-entries-dasar.md`  
    Tujuan: memakai built-in `Object` dasar untuk inspeksi data.
11. `C11-built-in-array-slice-concat-dan-join.md`  
    Tujuan: memakai built-in `Array` dasar yang aman untuk membaca dan menyusun ulang data.
12. `C12-built-in-array-includes-some-dan-every-dasar.md`  
    Tujuan: memahami built-in array dasar untuk pengecekan kondisi.
13. `C13-math-dan-number-builtins-praktis.md`  
    Tujuan: memakai built-in numerik dasar yang umum muncul di latihan pemula.
14. `C14-json-stringify-dan-parse-dasar.md`  
    Tujuan: memahami perpindahan data sederhana antara object JavaScript dan JSON.
15. `C15-object-array-dan-reference-behavior-di-function.md`  
    Tujuan: memahami perilaku object/array saat dikirim sebagai argumen function.
16. `C16-checklist-debugging-object-array-dan-builtins.md`  
    Tujuan: menyusun checklist debugging praktis untuk bug data structure dasar.

## Urutan Baca

- Wajib berurutan dari `C01` sampai `C16`.
- `C01-C04` adalah jalur fondasi object sebelum pembaca masuk ke daftar berurutan.
- `C05-C09` membangun intuisi array, iterasi, dan mutasi data.
- `C10-C14` memperkenalkan built-ins yang paling sering muncul dalam pekerjaan sehari-hari pemula.
- `C15-C16` merapikan model mental reference behavior dan pola debugging dasar.
