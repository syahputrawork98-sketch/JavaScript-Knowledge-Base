# SR-01: Core Entities (The Machine Core)

> **"SR-01 berisi komponen-komponen generator paling dasar yang mendefinisikan bagaimana objek, fungsi, dan nilai-nilai presisi beroperasi di dalam inti Hub Energi."**

Sub-Rak ini membedah "mesin utama" JavaScript yang sering kita gunakan tanpa menyadari kerumitan di bawahnya.

## Daftar Buku (Inventory)

### [BK-01: The Value Bases](./BK-01_ValueBases/README.md)
Blok bangunan dasar untuk semua struktur data.
- `Object`: Dasar dari segala sesuatu.
- `Function`: Mesin penggerak logika.
- `Boolean`: Saklar biner (True/False).
- `Symbol`: Identitas unik yang tidak bisa dipalsukan.

### [BK-02: Precision Core](./BK-02_PrecisionCore/README.md)
Pusat perhitungan dan manajemen nilai numerik.
- `Number`: Standar pengukuran energi.
- `BigInt`: Penanganan beban energi super besar.
- `Math`: Unit kalkulasi otomatis.
