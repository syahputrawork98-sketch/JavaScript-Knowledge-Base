# RAK-03: Standard Built-in Objects (The Energy Lab Inventory)

> **"Selamat datang di Gudang Inventaris Hub Energi. Di sini, Anda akan menemukan semua peralatan, kontainer, dan transformator standar yang disediakan oleh inti JavaScript untuk mengelola energi data Anda."**

RAK-03 adalah dokumentasi lengkap mengenai objek-objek bawaan (*Standard Built-in Objects*) yang tersedia secara global dalam lingkungan JavaScript (ECMAScript).

## Peta Inventaris (Hierarchy)

### [SR-01: Core Entities](./SR-01_CoreEntities/README.md)
Unit dasar dan mesin logika inti.
- **BK-01: The Value Bases**: `Object`, `Function`, `Boolean`, `Symbol`.
- **BK-02: Precision Core**: `Number`, `BigInt`, `Math`.

### [SR-02: Data Management](./SR-02_DataManagement/README.md)
Kontainer energi dan sistem penyimpanan.
- **BK-03: The Conveyor System**: `Array`, `TypedArray`.
- **BK-04: Labeled Storage**: `Map`, `Set`, `WeakMap`, `WeakSet`.

### [SR-03: Signal & Flow](./SR-03_SignalFlow/README.md)
Komunikasi teks, pola, dan sinkronisasi waktu.
- **BK-05: Communication Hub**: `String`, `RegExp`, `JSON`.
- **BK-06: Precise Syncing**: `Date`, `Intl`.

### [SR-04: Bridge & Buffer](./SR-04_BridgeBuffer/README.md)
Pengolahan data mentah tingkat rendah.
- **BK-07: Structured Buffers**: `ArrayBuffer`, `SharedArrayBuffer`, `Atomics`, `DataView`.
